Smart Majdoor Guidelines

Majdoor (Service Provider) Guidelines:

  1. Create an ID with the prefix "SMJ" (e.g., SMJ123).
  2. Provide all necessary personal information, skills, and experience.
  3. Provide Aadhar number for security purposes.
  4. Ensure all information is accurate and up-to-date.
  5. Upload a professional photo for customer's reference.
  6. Preview your profile to ensure it looks appealing to customers.
  7. Be responsive to customer inquiries and service requests.

Customer Guidelines:

  1. Create a unique ID with the prefix "CMD" (e.g., CMD456).
  2. Provide necessary personal information.
  3. Provide Aadhar number for security purposes.
  4. Browse available service providers based on requirements.
  5. Select a service provider based on skills, ratings, and reviews.
  6. Contact the selected service provider directly using provided contact details.
  7. Provide feedback after service completion to help improve the platform.
